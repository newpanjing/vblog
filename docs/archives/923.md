# Flutter在现有项目上启用macOS、Android、iOS、Windows等

<p>在当前项目的根目录中执行，记住后面有个小点</p>
<p>命令：</p>
<pre><code class="language-shell"> flutter create --platforms=macos .</code></pre>
<p>-- platforms换成其他的即可，可选：</p>
<ul>
<li>macOS</li>
<li>Windows</li>
<li>Android</li>
<li>iOS</li>
<li>web</li>
</ul>
