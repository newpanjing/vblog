# Django前后分离admin框架SimplePro发布1.4

# SimplePro v1.4 更新内容

- 修复 日期查询区间bug
- 修复 sessionStore可能引起的bug
+ 图表可通过配置显示隐藏
```python
# 配置Simple Pro是否显示首页的图标，默认为True，显示图表，False不显示
SIMPLEPRO_CHART_DISPLAY = True
```

- 表格筛选字段过多导致不显示问题


### 升级：

```shell
pip install simplepro -U
```

### 安装：

```shell
pip install simplepro
```

> 如果提示没有找到1.4版本，请切换至默认源进行更新。
